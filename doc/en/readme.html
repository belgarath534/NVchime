NVChime

NVChime lets you customize the sounds NVDA plays when it starts up and shuts down.

Recommended Setup

For the best experience, go to NVDA Settings, General, and uncheck "Play sound when starting or exiting NVDA" so only your NVChime sound plays.

Features

  • Six built-in sounds: Classic Chime, Retro Beep, Soft Bell, Dramatic Hit, Horror Sting, and Chill Tone
  • Use any WAV file from your computer
  • Separate startup and exit sounds
  • Adjustable startup delay
  • Preview sounds before saving

How to Use

  1. Open NVDA menu with NVDA+N
  2. Go to Preferences, then Settings
  3. Find the NVChime category
  4. Configure your sounds and hit OK
